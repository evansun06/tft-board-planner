<h2>Setup</h2>

- Requires Java Runtime Environment 8 or newer

```bash
# Clone
git clone https://github.com/evansun06/tft-board-planner.git
cd tft-board-planner

# Compile & Run
javac -d bin src/*.java
java -cp bin Main
```
